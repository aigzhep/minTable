"MinTable" plugin | Плагин "minTable"

Description | Описание

ENG:
This plugin allows to minimize a long table down to integer row number and to get short table view without any partial view of the last row.
To make a short-view of a table, you need to apply minimize() function to your table(-s): 
$(<identifier>).minimize({
  min: 0,
  hideText: "Hide",
  showText: "Show",
  fixHeader: false,
  initialView: "short"
});
Description of the parameters of the
minimize() function:
  • min - an amount of rows you want to show
  • hideText - a text inside a button (to hide rows) (the button is created automatically)
  • showText - a text inside a button (to show all rows)
  • initialView - an initial state of your table, short or full
  • fixHeader - fixation of <thead> table element, if it exists: true when it's needed to be always fully shown, false when it can be partially hidden
РУС:
Данный плагин позволяет сворачивать длинные таблицы до целого числа строк, чтобы в таком свернутом состоянии показывались целые строки, без частичного "обрезания" последней строки по высоте.
Для этого вам нужно привязать функцию minimize() к вашей(-им) таблице(-ам): 
$(<identifier>).minimize({
  min: 0,
  hideText: "Hide",
  showText: "Show",
  fixHeader: false,
  initialView: "short"
});
Описание параметров функции minimize():
  • min - количество видимых строк
  • hideText - текс внутри кнопки (при скрытии) (кнопка генерируется автоматически)
  • showText - текст внутри кнопки (при раскрытии)
  • initialView - начальный вид таблицы, short (свернута) или full(полностью показана)
  • fixHeader - фиксация элемента <thead> таблицы, если таковой есть: true - он будет видимым всегда, false - строки из него можно будет скрывать.
